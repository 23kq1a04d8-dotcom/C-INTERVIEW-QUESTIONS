1. What is a static variable in C?

Answer: A static variable retains its value between function calls and is initialized only once.


---

2. What is an auto variable?

Answer: An auto variable is a local variable created automatically when a function starts and destroyed when it ends.


---

3. What is the difference between ++i and i++?

Answer:

++i (pre-increment): Increments first, then uses the value.

i++ (post-increment): Uses the value first, then increments.



---

4. What is a file in C?

Answer: A file is used to store data permanently on secondary storage.


---

5. Which functions are used to open and close a file?

Answer:

fopen() – Opens a file.

fclose() – Closes a file.



---

6. What is the purpose of scanf()?

Answer: scanf() is used to read input from the user.


---

7. What is the purpose of printf()?

Answer: printf() is used to display output on the screen.


---

8. What is an infinite loop?

Answer: An infinite loop is a loop that never ends because its condition always remains true.


---

9. What is the difference between char and char[]?

Answer:

char stores a single character.

char[] stores a string (multiple characters).



---

10. What is the entry point of every C program?

Answer: The main() function is the entry point of every C program.
