1. What is a pointer in C?

Answer: A pointer is a variable that stores the memory address of another variable.


---

2. What is a NULL pointer?

Answer: A NULL pointer does not point to any valid memory location.


---

3. What is the difference between malloc() and calloc()?

Answer:

malloc() allocates memory but does not initialize it.

calloc() allocates memory and initializes it to zero.



---

4. What is a structure (struct) in C?

Answer: A structure is a user-defined data type that groups variables of different data types under one name.


---

5. What is a union in C?

Answer: A union is a user-defined data type where all members share the same memory location.


---

6. What is the difference between a structure and a union?

Answer:

Structure: Each member has separate memory.

Union: All members share the same memory.



---

7. What is a segmentation fault?

Answer: It is a runtime error that occurs when a program accesses invalid memory.


---

8. What are command-line arguments?

Answer: They are values passed to the program when it is executed from the command line.


---

9. What is the use of the sizeof operator?

Answer: It returns the size (in bytes) of a variable or data type.


---

10. What are header files in C?

Answer: Header files contain function declarations, macros, and definitions that can be included using #include.
