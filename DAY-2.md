DAY-2: C Interview Questions

1. What is an array in C?

Answer: An array is a collection of elements of the same data type stored in contiguous memory locations.


---

2. What is the difference between while and do-while loops?

Answer: while checks the condition before execution, while do-while executes the loop at least once before checking the condition.


---

3. What is a function in C?

Answer: A function is a block of code that performs a specific task and can be called whenever needed.


---

4. What is the purpose of the return statement?

Answer: It ends the function execution and returns a value to the calling function.


---

5. What is recursion?

Answer: Recursion is a process where a function calls itself until a base condition is met.


---

6. What is the difference between a local and a global variable?

Answer: A local variable is declared inside a function and is accessible only within that function. A global variable is declared outside all functions and can be accessed throughout the program.


---

7. What is a string in C?

Answer: A string is an array of characters terminated by the null character ('\0').


---

8. What is the break statement used for?

Answer: It immediately terminates a loop or switch statement.


---

9. What is the continue statement used for?

Answer: It skips the current iteration and continues with the next iteration of the loop.


---

10. What is the difference between = and ==?

Answer:

= is the assignment operator.

== is the comparison operator used to check equality.
