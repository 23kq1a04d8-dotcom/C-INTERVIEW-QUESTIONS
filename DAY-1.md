1. What is C?

Answer: C is a general-purpose, procedural programming language.

2. Who developed C?

Answer: Dennis Ritchie developed C in 1972 at Bell Labs.

3. Why is C called a middle-level language?

Answer: Because it supports both low-level (hardware) and high-level (application) programming.
4.What is the main() function?

Answer: It is the entry point where the execution of a C program begins.
5.What is a variable?

Answer: A variable is a named memory location used to store data.
6.What is the difference between a variable and a constant?

Answer: A variable can change , while a constant cannot during program execution.
7.What is the difference between printf() and scanf()?

Answer: printf() is used to display output, while scanf() is used to take input from the user.
8.What is a loop in C?

Answer: A loop is used to execute a block of code repeatedly until a condition becomes false.
9.What is the purpose of the continue statement?

Answer: It skips the current iteration and continues with the next iteration of the loop.

10. Which loop executes at least once? why?
Answer: The do-while loop executes at least once because it checks the condition after executing the loop body.
